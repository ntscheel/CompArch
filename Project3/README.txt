TEST CASES:
The test cases directory contains assembly code for five test files. They are as follows:

BranchNotTaken.txt - tests a BEQ instruction that should not be taken.
BranchTaken.txt - tests a BEQ instruction that should be taken.
DataHaz1.txt - tests multiple consecutive LW and ADD instructions to test data hazards.
DataHaz2.txt - identical to DataHaz1, but includes an SW instruction.
Proj1HandoutCode.txt - the test code given for project 1 - tests loops.
